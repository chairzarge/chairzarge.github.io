Module #7 (Week 6) - JavaScript Objects

Description:

This week's learning focuses on Javascript objects, specifically using the Document Object Model (DOM) and native objects like Math, String, Array, etc. This module works on defining and creating objects, using built-in objects, and combining objects and arrays to implement functions.

Pages:

    Practice Exercises:
    • Exercise 1: Create an object called Person
    • Exercise 2: Use the DOM to show the main information of a page
    • Exercise 3: Use the STRING object to provide different formats to a text
    • Exercise 4: Use the NUMBER object to validate your input
    • Exercise 5: Use the MATH object generate random numbers
    • Exercise 6: Use the DATE object to add a date/time to your portfolio
