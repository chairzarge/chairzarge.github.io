Module #1 (Week 1) - Introduction to Web Development: HTML

Description:

After establishing a website (in this case chairzarge.github.io) in Module #0, Module #1 fills in with how to actually make use of the website with basic html. The lecture goes over an html document that consists of tags that format the document (for instance, the <html> and </html> tags mark where in the file begins the html and the <head> </head> marks where you put the metadata for the document).

After establishing the fundamentals, the module provides practice (see below) and other assignments.

Pages:

    Practice Exercises:
    1. Exercise 1: Create your first page: "Hello World"
    2. Exercise 2: Create a page that displays a recipe
    3. Exercise 3: Modify Exercise 2 and include images and links to similar recipes
    4. Exercise 4: Use relative and absolute links
