Module #3 (Week 3) - Introduction to CSS: Color, Text, Boxes, List, Tables & Forms

Description:

This week we are presented the new idea of CSS which is used to style your html by using declarations within element selectors. These declarations consist of a "<property> : <value>" pair. Examples include the "color" property which can have a string, RGB, hex, or HSL value; the "font-family" property, which determines the type of font used for the element text; and plenty of others.
