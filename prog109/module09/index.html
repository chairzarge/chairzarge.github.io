Module #9 (Week 8) - Javascript Modules and Events

Description:

This week's learning module focuses on using the DOM and JavaScript functions to handle trigger events, such as clicking on an item or mousing over an object. I will learn to define events, bind them to DOM objects, trigger events using JavaScript functions, and monitor events using delegation. By the end of the module, I should be able to create JavaScript functions, code event-driven procedures, and use objects and web forms effectively.
